title: 接口地址内网改造
---

- 外网前缀地址：`https://dev-gateway.chinamarket.cn/gateway/api`
- 如商品地址：`https://dev-gateway.chinamarket.cn/gateway/api/goods-web`

- 内网前缀地址：`http://mall-gateway-web-svc:8080/api/`
- 如商品地址：`http://mall-gateway-web-svc:8080/api/goods-web`

## 对应表格

| 接口标题 | 外网地址 | 测试服内网 | 正式服内网 |
| --- | -- | -- | -- |
| 商品 | `/goods-web` | `http://mall-gateway-web-svc:8080/api/goods-web` | `http://mall-goods-web-svc:8080/api/goods-web` |
| 用户 | `/user-web` | `http://mall-gateway-web-svc:8080/api/user-web` |  |
