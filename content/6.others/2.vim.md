## 介绍

Vim 是一个功能强大的文本编辑器，广泛用于开发和编辑代码。

## 基本操作

### 1. 启动 Vim

```bash
vim filename
```

### 2. 基本模式

- 按 `Esc` 键确保处于命令模式
- `i` 进入插入模式
- `Esc` 退出插入模式
- `:w` 保存文件
- `:q` 退出 Vim
- `:wq` 保存并退出
- `:q!` 不保存退出

### 3. 常用快捷键

- `yy` 复制当前行
- `p` 粘贴
- `dd` 删除当前行
- `u` 撤销
